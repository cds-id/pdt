Key Design Decisions

Session Storage

whatsmeow device store saved as blob in wa_numbers.device_store — survives server restarts without file-based storage

Outbox Pattern

wa_outbox holds agent-drafted messages with approval status. Agent sets pending, user approves/rejects from UI, sender worker picks up approved items

Media Handling

Media files uploaded to R2, metadata in wa_media. Same pattern as existing report file uploads

Weaviate Sync

On message insert → async embed + push to Weaviate. If Weaviate is down, messages still saved in MySQL — embeddings catch up later
