WhatsApp Agent Tools (7)

1. list_listeners

List all registered listeners across all numbers. Returns name, type (group/personal), number, active status, message count.

Params: wa_number_id? (optional filter)
2. search_messages

Keyword search across messages. Filters by listener, sender, date range. Returns matching messages with context.

Params: query, listener_id?, sender?, start_date?, end_date?, limit?
3. semantic_search

Vector similarity search via Weaviate. Finds messages by meaning, not just keywords. Great for "what did we discuss about X?"

Params: query, listener_id?, start_date?, end_date?, limit?
4. summarize_chat

Generate AI summary for a listener or across all listeners in a time range. Key topics, action items, decisions.

Params: listener_id? (null = all), start_date, end_date
5. send_message

Draft an outgoing message for approval. Creates wa_outbox entry. Agent provides content + context (why it wants to send).

Params: target_jid, content, context
6. reply_to_message

Draft a reply to a specific message in a listener. Creates wa_outbox with reply context. Quoted reply in WhatsApp.

Params: wa_message_id, content, context
7. full_chat_report

Compound tool — single call that runs: list_listeners + summarize_chat (all) + semantic_search (recent topics). Returns complete WhatsApp briefing.

Params: start_date, end_date

Agent System Prompt Highlights

  • Anti-hallucination: Same rules as BriefingAgent — NEVER fabricate message content, only present tool results
  • Self-awareness: Knows the current user's phone numbers, distinguishes user's own messages from others
  • Send discipline: Must always explain WHY it wants to send a message (context field). Never sends without user approval.
  • Summarization with vector context: For summaries, agent first does semantic_search to find relevant threads, then uses those as context for richer summaries
  • Cross-agent potential: Orchestrator can route follow-ups to Jira/Git agents (e.g., "someone mentioned ticket PDT-123 in chat" → JiraAgent lookup)
